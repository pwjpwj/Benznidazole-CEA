# Benznidazole CEA
 Cost-effectiveness study of benznidazole for the treatment of Chagas disease

 This is a cost-effectiveness evaluation of benznidazole for the treatment of chronic indeterminate Chagas disease. The model is programmed in R and RStudio is recommended for execution.
 
# Model Calibration
First execute BZN_Calib_01, then BZN_Calib to evaluate graphically the best parameters.

Execute BZN_CEA_01

# OWA
Execute BZN_OWA

# PSA
